Mark Sex

The kin term in question may have the wrong sex marking. If the term should have the sex indicated by the question, select 'Yes' and the KAES program will change the sex of the indicated kin term in the kin term map. The change will only affect the scratch file used to construct the kin term map and not the data file for the kin term map. If the data file should be modified as well, use the Save option under the Files option to replace the data file by the modified scratch file.
